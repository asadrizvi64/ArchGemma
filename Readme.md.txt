# Gemma Model Fine-Tuned on LLaMA Synthetic Architecture Q/A Pairs

This repository contains the code for a deep learning project, which fine-tunes the Gemma model using LLaMA-generated synthetic architecture question-answer pairs.

## Model Weights
The model weights are hosted on Hugging Face.

- [Model on Hugging Face](https://huggingface.co/your-username/your-model)

## Usage
To use this model, download the weights from Hugging Face and place them in the `model/` directory.
